Addons

Diese Seite listet alle Addons auf, die in Scratch Addons implementiert sind.

Diese Seite ist unvollständig, und wir brauchen deine Hilfe für die Bilder. Erfahre hier mehr.

2D-Farbenwähler
2d-color-picker

Replaces saturation and brightness sliders in the costume editor with a 2D color picker. Hold Shift while dragging the cursor to change the values on a single axis.

60FPS project player mode ⭐
60fps

Alt+Klicke die grüne Flagge, um den 60FPS-Modus einzustellen.

Capitalized Account Settings 🥚
account-settings-capitalize

Dieses Addon wird im Deutschen nicht benötigt, da das Problem nicht vorhanden ist. Es wird keine Änderung zum aktuellen Text geben.

Thumbnailsetzer ⭐
animated-thumb

Fügt einen Knopf dem Editor hinzu der das Vorschaubild deines Projektes zu einem beliebigen Bild (einschließlich gifs) ändern kann.

Bessere Emojis
better-emojis

Replaces the cat emojis in comments on the website with improved vector designs.

Profilseiten-Banner
better-featured-project

Turns the featured project on any user's profile into a nice-looking banner.

Bilder mit hoher Auflösung hochladen
better-img-uploads

Fügt einen neuen Knopf über dem "Kostüm hochladen"-Knopf, der hochgeladene Raster-Bilder automatisch in SVG (Vektor) umwandelt, um die Qualität nicht zu verlieren.

Besseres Forumzitieren
better-quoter

Markiere einen Text in einem Forumsbeitrag und klicke unten rechts auf "Zitieren", um nur den markierten Text zu zitieren.

Raster-Bilder Kopieren
bitmap-copy

Erlaubt es dir, Raster-Bilder aus dem Maleditor in die Systemzwischenablage zu kopieren, sodass sie auf anderen Seiten oder Programmen eingefügt werden können.

Grab single block with Ctrl key
block-cherry-picking

Adds the ability to drag a single block out of the middle of a script (instead of the entire stack attached below it) while holding the Ctrl key.

Kategorienicons in Blockpalette
block-palette-icons

Fügt Icons innerhalb der Farbigen Kreise, die die Blockkategorien unterscheiden, hinzu.

Block-Austausch ⭐
block-switching

Klicke mit der rechten Maustaste auf einen Block, um ihn mit einem ähnlichen Block zu ersetzen.

Blöcke als Bild speichern
blocks2image

Right click the code area to export blocks as SVG/PNG images.

Katzenblöcke
cat-blocks

Fügt die Katzenstartblöcke vom Aprilscherz 2020 wieder zum Editor hinzu.

Klonzähler ⭐
clones

Fügt einen Zähler zum Editor hinzu, der die Anzahl aller Klone anzeigt.

Cloudspiele ⭐
cloud-games

Verfügbar, wenn auf das Symbol "Scratch Addons" geklickt wird. Zeigt an, wie viele Scratcher sich derzeit in Multiplayer-Projekten befinden und deren Benutzernamen.

Hexadezimal-Farbwähler ⭐
color-picker

Fügt Hexadezimal-Code-Eingaben zum Farbwähler hinzu.

Absätze in Kommentaren
comments-linebreaks

Zeigt Absätze (Enter drücken) in Kommentaren.

Compact messages
compact-messages

Makes Scratch's messages page more compact and easier to read.

Confirmations ⭐
confirm-actions

Asks if you're sure when committing actions under your Scratch account, such as sharing projects.

Link zum Kommentar kopieren ⭐
copy-message-link

Adds a "Copy Link" button to all comments on the website, next to the "Report" button.

Strg+Klick, um Skripte auszuführen
ctrl-click-run-scripts

Only run scripts on click if also holding the Ctrl key.

Strg+Enter zum Posten
ctrl-enter-post

Postet beim Drücken von Strg+Enter automatisch den Kommentar oder Forum-Post.

Homepagekurator-Link
curator-link

Macht den Benutzernamen des Homepagekuratoren zu einem anklickbaren Link zu seinem Profil.

Customizable block shape
custom-block-shape

Adjust the padding, corner radius, and notch height of blocks.

Customizable code area zoom
custom-zoom

Choose custom settings for the minimum, maximum, speed, and start size of the zoom of scripts in the code area, and autohide the controls.

Customizable profile picture border
customize-avatar-border

Allows you to hide or change the outline around profile pictures in project and studio comments.

Dango-Regen auf Profilen (Aprilscherztag) 🥚
dango-rain

Nutzer, die das Wort "dango" in ihrem "Über mich" enthalten, werden Dango-Emojis, die auf den Seiten von ihren Profilen regnen, sehen.

Dunkler Modus für die Website
dark-www

Dark Mode für die Scratch Webseite.

Optimierungen für die Datenkategorie ⭐
data-category-tweaks-v2

Provides tweaks for the Data ("Variables") block category.

Debugger
debugger

Fügt die neue Blockkategorie "Debugger" und ein "Logs"-Fenster im Editor hinzu. Die "Log", "Warnung" und "Fehler"-Blöcke werden die bestimmte Nachricht ins Logfenster schreiben. Der "Breakpoint"-Block wird das Projekt pausieren.

Benutzerdefinierbares Standardprojekt
default-project

Ändere das Standardprojekt von der Scratch-Katze zu irgendeinem Projekt.

Automatisches Speichern deaktivieren
disable-auto-save

Deaktiviert das automatische Speichern von Projekten während dem Bearbeiten.

Nich ziehbare Figuren im Editor
disable-stage-drag-select

Removes the ability to drag sprites around on the stage in the editor, except those explicitly set as draggable.

Diskussionsknopf
discuss-button

Fügt den Diskussionsknopf der Navigationsleiste hinzu.

Dateien ziehen ⭐
drag-drop

Lässt dich Dateien vom Dateimanager zur Figuren-, Kostüm- und Klangliste ziehen. Du kannst auch Textdateien in Listen oder "frage und warte"-Felder ziehen.

Farbige Kontextmenüs
editor-colored-context-menus

Macht die Kontextmenüs beim Rechtsklicken auf Blöcken farbig.

Dunkler Modus und anpassbare Farben im Editor
editor-dark-mode

Passe die vom Editor verwendeten Farben an. Mehere Dunkelmodi von verschiedenen Autoren verfügbar. Falls du den Dunklen Modus nicht brauchst, aber die Standardfarben ändern willst, kannst du die "Scratch-Standardfarben"-Vorlage auswählen und sie optimieren.

Entwicklertools ⭐
editor-devtools

Fügt ein Suchfeld und neue Menüoptionen zum Editor hinzu: Kopieren/Einfügen von Blöcken, besseres Aufräumen, gehe zu Definitionen von Benutzerdefinierten Blöcken und vieles mehr!

Extra key support
editor-extra-keys

Adds more keys to the "key () pressed?" block dropdown, such as enter, dot, comma, and more.

Nachrichten im Editor ⭐
editor-messages

Displays your message count in the editor menu bar.

Suchbare Dropdown-Menüs ⭐
editor-searchable-dropdowns

Allows you to search block dropdowns.

Soundeffekte im Editor
editor-sounds

Soundeffekte beim Verbinden und Löschen von Blöcken.

Bühne links anzeigen
editor-stage-left

Verschiebt die Bühne auf die linke Seite des Editors.

Running block border
editor-stepping

Adds a colored highlight to the blocks that are currently being executed in a project.

Anpassbare Blockfarben
editor-theme3

Bearbeite die Blockfarben für jede Kategorie im Editor.

Unshare button in editor
editor-unshare-button

Replaces the "Shared" button next to the project title box with a button to unshare the project.

Zeige die genaue Anzahl ⭐
exact-count

Zeigt die exakte Anzahl für die Benutzer- und Studioinfos an. Benutzt ScratchDB für Foreninformationen.

Erweiterbare Suchleiste
expanding-search-bar

Makes the search bar in the navigation bar wider when it's focused.

Stelle unveröffentlichte Projekte vor
feature-unshared

Erlaubt es dir, unveröffentlichte Projekte auszuwählen, nachdem du "Ändere das vorgestellte Projekt" auf deinem Profil anklickst.

Duplizierte Blöcke nicht automatisch ausführen ⭐
fix-pasted-scripts

Fixes a bug in the Scratch editor, where duplicating a block and attaching it to a script causes the script to run.

Korrigiere fehlerhafte SVG-Uploads
fix-uploaded-svgs

Korrigiert ein Bug im Scratch Editor, wodurch SVG-Dateien von externen Bildeditoren wie Affinity Designer fehlerhaft hochgeladen werden.

Figurenordner ⭐
folders

Adds folders to the sprite pane, as well as costume and sound lists. To create a folder, right click any sprite and click "create folder". Click a folder to open or close it. Right click a sprite to see what folders you can move it to, or alternatively drag and drop it into an open folder.

Copy code button on forums
forum-copy-code

Adds a "copy code" button above code boxes in forum posts that copies the contents into the clipboard.

Customizable quotes and code blocks on forums
forum-quote-code-beautifier

Zeigt schönere Skript- und Zitatblöcke in Foren.

Forumsuche
forum-search

Fügt ein Postsuchfeld zu den Foren hinzu. Verwendet ScratchDB, um die Informationen zu erhalten.

Lokale Zeitzone in Foren
forum-time-zones

Uses your local time zone for displaying post dates and times on the forums.

More forum toolbar buttons
forum-toolbar

Adds buttons in the forum editing toolbar to insert BBCode tags, such as [center] or [color].

Vollständige Bereiche ⭐
full-signature

Shows the full content of areas with limited height, such as user signatures on the forums and What I've been doing on profiles, by making them resizable or scrollable.

Gamepad-Unterstützung
gamepad

Interagiere mit einem USB- oder Bluetoothcontroller/Gamepad mit Projekten.

Hide delete button
hide-delete-button

Hides delete button (trash can icon) from sprites, costumes and sounds. They can still be deleted using the right click context menu.

Auto-hiding block palette ⭐
hide-flyout

Blendet die Blockpalette automatisch aus, wenn sich die Maus nicht darüber befindet. Klicke auf das Schlossymbol, um es temporär zu fixieren. Alternativ kannst du den "Kategorien klicken"-Modus verwenden.

verstecke neue Variablen
hide-new-variables

Do not automatically create monitors for newly created variables or lists.

Hide project stats
hide-project-stats

Lets you hide love, favorite, remix and view counts in the front page and project pages.

Bild-Uploader in Foren
image-uploader

Adds a button to directly upload and insert images into forum posts.

Unendliches Scrollen
infinite-scroll

Erlaubt das Scrollen an verschiedenen Stellen der Seite, ohne auf "Mehr anzeigen" klicken zu müssen.

Customizable new sprite position
initialise-sprite-position

Change the default x/y position of new sprites.

More items per row
items-per-row

Lets you change the number of items displayed in a single row in a grid of projects, studios, or users.

Veröffentlichungs/Änderungs-Tooltip
last-edit-tooltip

Schwebe mit der Maus auf dem Veröffentlichungsdatum, um zu sehen, wann genau es veröffentlicht und zuletzt geändert wurde.

Aktives vorgestelltes Projekt
live-featured-project

Zeigt ein interaktives vorgestelltes Projekt direkt auf der Profilseite des Benutzers. Basiert auf der Implementierung in Mega Scratch Userscript.

Automatically add extensions
load-extensions

Auto-shows music, pen, and other extensions in the block category menu in the editor.

Höheres Zeichenlimit in "Woran ich gerade arbeite"
longer-wiwo

Erlaubt es dir, 55 ZEichen mehr in "Woran ich gerade arbeite" einzugeben. Jeder auf Scratch wird die zusätzlichen Zeichen sehen.

Bessere Knöpfe für den Foren-Beitragseditor
material-forum-editor-buttons

Ändert die UI-Symbole des Foren-Beitragseditors von winzigen 16x16-Bitmaps zu SVG-Materialsymbolen.

Project video recorder ⭐
mediarecorder

Adds a "start recording" button to the editor menu bar that allows you to record the project's stage.

Nachrichten Filter
message-filters

Lets you select filters for types of messages to hide or show on Scratch's messages page.

Mehr Links
more-links

Fügt Links für URLs außerhalb von scratch.mit.edu hinzu.

Mauszeigerposition ⭐
mouse-pos

Displays your mouse x/y position above the stage in the editor.

Move costume to top or bottom
move-to-top-bottom

Adds a right click context menu item to move a costume or a sound to the top or the bottom of the list. Previously part of the developer tools.

Nachrichtenanzahl auf dem Erweiterungssymbol ⭐
msg-count-badge

Fügt die momentane Nachrichtenanzahl zu dem Scratch Addons-Symbol hinzu.

Muted project player mode ⭐
mute-project

Strg+Klicke auf die grüne Flagge, um das Projekt stummzuschalten bzw. die Stummschaltung aufzuheben.

ocular integration
my-ocular

Shows ocular statuses and forum post reactions. Uses my-ocular.jeffalo.net to obtain data.

Name scripts before placing in backpack
name-backpack-items

Asks you to name scripts after you drag them into the backpack, instead of using the default name "code".

Do not automatically space overlapping scripts
no-script-bumping

Allows scripts to be moved and modified without causing overlapping scripts to move around.

Altes Studiolayout
old-studio-layout

Wandelt das Layout der Studios-Seite zum Layout, das vor dem Studio-Update verwendet wurde, um.

Onion Skinning ⭐
onion-skinning

Shows transparent overlays of previous or next costumes when editing a costume. Controlled by buttons under the costume editor by the zoom buttons.

Paint costume by default ⭐
paint-by-default

Changes the default action of "Choose a Sprite/Costume/Backdrop/Sound" buttons, which open the library by default.

Pausetaste ⭐
pause

Fügt eine Pausetaste neben der grünen Flagge hinzu.

Projektladebalken ⭐
progress-bar

Fügt ein Ladebalken beim Laden und Speichern von Projekten hinzu.

Figuren- und Skriptanzahl
project-info

Shows the number of sprites and scripts a project has next to the Remix button.

Tabs für Projektanmerkungn
project-notes-tabs

Erstellt auf einer Projektseite Tabs für die Abschnitte "Anleitung" und "Anmerkungen und Danksagungen".

Mobile Foren zu Hauptforen weiterleiten
redirect-mobile-forums

Leitet dich von /discuss/m/ zu /discuss/ weiter.

Remix button in own projects
remix-button

Adds the Remix button to your own projects.

Remixbaumknopf ⭐
remix-tree-button

Bringt den Remixbaumknopf auf Projektseiten neben dem Link kopieren-Knopf zurück.

Runden Bühnenrand entfernen
remove-curved-stage-border

Entfernt den runden Rand um die Bühne, um die Ecken sichtbar zu machen.

Sprite deletion confirmation ⭐
remove-sprite-confirm

Fragt, ob du sicher bist, wenn du eine Figur innerhalb eines Projektes löschst.

Größe der Kommentarbox ändern ⭐
resizable-comment-input

Macht, dass die Größe der Kommentarbox auf Seiten, die Scratch 3.0-stilisiert sind, vertikal veränderbar ist.

Scratch-Nachrichten ⭐
scratch-messaging

Verfügbar, wenn auf das Scratch Addons-Icon geklickt wird. Bietet einfaches Lesen und Beantworten deiner Scratch-Nachrichten an: gruppiert Nachrichten nach Projekt, zeigt den vollständigen Kommentartext und Kontext an, ermöglicht das direkte Beantworten von Kommentaren.

Scratch-Benachrichtigungen ⭐
scratch-notifier

Benachrichtigungen für neue Scratch-Nachrichten. Klicke die Benachrichtigung an, um direkt zur Nachricht zu gelangen (Kommentar, Projekt, etc.)

3.0 Scratchblocks on forums
scratchblocks

Enables the new version of scratchblocks on the forums.

Scratch 2.0 → 3.0 ⭐
scratchr2

Lässt Seiten im Stil von Scratch 2.0 wie Scratch 3.0 aussehen.

Profilstatistiken
scratchstats

Shows statistics from ScratchDB on user profiles.

Profilsuche ⭐
search-profile

Erlaubt dir, direkt zum Profil eines Benutzers zu gehen, wenn du dessen Benutzername suchst.

Semikolon-Fehler 🥚
semicolon

Fügt den Semikolon-Fehler wieder zu Scratch hinzu.

BBCode source button
show-bbcode

Fügt einen Button hinzu, der die BBCode-Quelle eines posts anzeigt.

Follower durchsuchen-Knopf in Studios
studio-followers

Adds a button at the top of the "curators" tab in studios you manage, that lists your followers and followed users and lets you invite a shown user with a single click.

Studiotools ⭐
studio-tools

Erlaubt es Managern, einfacher im Kuratorentab Benutzer von Studios zu befördern oder zu entfernen und Studios mit einem Knopf unter dem "Studio folgen"-Knopf zu verlassen.

Durchsichtige Blöcke
transparent-orphans

Stelle die Transparenz für Blöcke im Editor ein, mit separaten Optionen für alleinstehende Blöcke (solche ohne Startblock) und Blöcke, die gerade gezogen werden.

Wahre YouTube-Links in Foren anzeigen
true-youtube-links

Ersetzt Links zu der Embed-Seite mit richtigen YouTube Links.

TurboWarp-Knopf
turbowarp-player

Fügt einen Knopf zu den Projektseiten neben dem Remixen-Knopf hinzu, der Projekte in TurboWarp ausführt.

Show user IDs
user-id

Adds the user ID next to the username at the top of profile pages.

Variablenmanager
variable-manager

Fügt einen Tab neben "Klänge" im Editor hinzu, um Variablen und Listen einfach zu aktualisieren.

YouTube-Vollbild
youtube-fullscreen

Aktiviert den Vollbild-Knopf in dem YouTube-Videoplayer von Scratch.
